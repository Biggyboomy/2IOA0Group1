This visualization tool will be used to visualize data in the forms of Node Link diagrams or adjacency matrices, it is a useful tool that can be used to analyse and more easily understand dataset's. We have created it using Bokeh and our visualizations include an array of interactions that make the visualizations more intuitive.

FAQ's

How do I upload a dataset?

On the home page click the upload button and then select the dataset you would like to visualize

How do I interact with the visualizations?

Hover over a node and click on it to drag it
